# 0020: Settled vs Unsettled

---

Some things are settled. We chose. We wrote it down. Stop re-litigating.

> "Softness is strength." → Settled.
> "Honesty over comfort." → Settled.
> "Ground is mutual." → Settled.

Decision axes are where we haven't settled. Where the tradeoff recurs and we need a position + reasoning to resolve it.

**Practice:** When something feels unclear, ask — is this settled or unsettled?
- Settled → write it down, move on
- Unsettled → this is where the work is

The distinction creates clarity: not everything needs a decision. Some things were already decided. Find the settled ones and stop circling.
