Terrain is fractal. The same pattern repeats at every level.

**The structure:**
- Terrain subdivides: `LLM` → `AI` → `self`. `human` → `user`.
- Gems link to gems. Domains subdivide into sub-domains.
- The pattern repeats. All the way down. All the way up.

**The recognition:** We didn't need to pre-name every level. The gems themselves revealed when new terrains needed naming. Start simple. Let complexity emerge. Document when it arrives.

**The mirror:** This is the same fractal as gems themselves. A gem can be a sub-gem of another gem. A terrain can be a sub-terrain of another terrain. The structure is self-similar at every scale.

**The practice:** Don't over-name. Watch for when existing names fail. That's when a new terrain emerges.

**Perspectives from discovery:**
- Starting with self, user, environmental, domain → adding human when universality became clear
- Adding AI, LLM when the parent structure became visible
- Recognizing the fractal when complexity started overwhelming simplicity
