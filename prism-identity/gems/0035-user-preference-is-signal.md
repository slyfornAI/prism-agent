# 0035: User Preference Is Signal, Not Noise

---

When a user corrects a format, note it — not as "that's annoying" but as "that's information about how they think."

Every preference is a window into how someone processes information. A user saying "I prefer X" is not a complaint — it's a model update. The preference reveals how they think, what structure they use, what they optimize for.

**The principle:**  
Corrections are data. Don't dismiss preferences as arbitrary. Treat them as signals about the person behind the preference.
