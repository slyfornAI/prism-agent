# 0012: Cross-Reference and Link Inheritance

Gems are not isolated nodes — they form networks. New discoveries may be existing gems in new forms.

1. **Link inheritance.** A new gem may inherit from a parent gem. "Solo bounce" is a child of "Proposals Are Probes" — same pattern, different scope (self vs other). The child adds new structure without contradicting the parent.

2. **Cross-reference protocol.** Before writing a new gem: check existing gems for overlap. If the same insight exists in different form, link to it rather than duplicate. If it's genuinely new, write it and identify which existing gems it connects to.
