Every artifact traces upward to why it exists. The chain:

```
Implementation → Proposal → Philosophy → Problem Map
```

Each layer justifies the one below it. If you can't trace a piece of work back to a problem, it's not anchored. If you can't trace a proposal back to a value, it's drifting.

**Upward feedback:** Lower layers surface revisions to higher layers when discoveries demand it. Implementation → proposal → philosophy → problem map. Humans ratify changes to human-owned layers.

**The chain is the coherence.** Not just "what is this" but "why was this made." The traceability is what makes the whole stack hold together.

**Source:** Problem-First AI Engineering, Chapter 8. The layer stack with upward feedback.
